# 壁纸图片分类体系

本项目分类按照下面核心原则组织：

- 分类：内容主体（唯一主分类）、内容细分（二级分类）
- 标签：情绪 / 风格 / 颜色

其中文件夹组织方式：

```
- 一级分类A
 - 子类文件夹A1
  - 20260606 # 按日期组织图片
    - p1.png  # 原图
    - p1.webp # 缩略图
    - p1.json   # 图片属性描述，比如情绪、风格、颜色、场景、提示词、作者等
    - p2.png
    - ...
 - 子类文件夹A2
- 二级分类B
 - 子类文件夹B2
- ...

```

每一张图都会对应一个大分类-子分类-日期，包含原图(png/jpeg)格式和webp缩略图格式，以及结构化json描述，方便AI读取解析入库。
